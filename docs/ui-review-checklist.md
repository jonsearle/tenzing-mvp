# UI Review Checklist

Use this on every screen before calling the UI done.

The aim is not perfection.
The aim is to catch the common things that make product UI feel cluttered, generic, or hard to use.

## Core Check

- Is the screen's main purpose obvious within a few seconds?
- Is there one clearly dominant action or focal area?
- Can anything be removed without harming understanding or task completion?

## Layout

- Does the layout fit the content type: list, table, form, or detail view?
- Are we using spacing and alignment to create structure before adding cards or borders?
- Are any cards being used just as layout containers rather than for real entities?
- Is there any unnecessary nesting of panels, sections, or containers?

## Text

- Is the page title clear and short?
- Does any intro text repeat what the title or UI already makes obvious?
- Are labels, button text, and helper text concise?
- Is there any text on screen that explains a confusing design instead of fixing it?

## Visual Restraint

- Are typography styles limited and consistent?
- Is bold text used for meaning, not decoration?
- Are icons only present where they add meaning?
- Is colour being used to signal action or status, not just to add visual interest?
- Are there any gradients, decorative backgrounds, or other non-functional visual flourishes we could remove?

## Interaction

- Is the primary action obvious?
- Are secondary and destructive actions visually subordinate?
- If there are multiple actions, is it still clear what the user should do next?

## Data Display

- If this is a collection of records, would a table or list work better than cards?
- If cards are used, is each card a meaningful unit the user reads or acts on as one thing?
- Are we showing only the information needed now, rather than everything available?

## Final Gut Check

- Does the screen feel calmer after simplification?
- Does anything still feel like default AI-generated SaaS UI?
- If we removed 20% of the elements, what would go first?

## Fast Pass

If time is tight, ask only these five questions:

1. What is this screen for?
2. What is the main action?
3. What can we remove?
4. Are cards or borders doing work that spacing should do?
5. Is anything visually loud without being important?
